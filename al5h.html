Frequently Asked Questions for AL5H Locks:

L1) How to Reprogram a lock as a guest or common door?


Large Keypad w/ Slider (StandAlone Unit) Instrctions
Detach the Keypad from Slider

Small Keypad (legacy) Instructions

Touch Screen Tablet Instructions
Double Click DTU Starter Icon
Tap Ok [F1] Button
Enter Staff Code and Password
Tap Ok

For Guests Room:
Select 1. Lock Setup Menu
Select 1. Initialize Lock Data
For Room Type Enter 2
Enter the Room #
Enter the Floor # (Example: for first floor, enter 01) then press Ok
Press Download
(Make sure to wake up the lock sensors by inserting any card, leaving it there and having it light up red first)
Press Ok
Aim the Infrared USB at the lock

For Common Doors:
After Entering Staff Code and Password-
Select 1. Lock Setup Menu
Select 1. Initialize Lock Data
For Room Type Enter 1
For Room # (Depends on which floor it is. If first floor put in 101, or else second floor- 201, etc)
(Room # isn't important for common doors, you can have multiple common doors w/ same #)
Then Press Ok Enter the Floor # (Example for first floor enter 01)
Dobule Tap the numbers to the right of room range 1
On Room Range 1, change FROM 000000 to the lowest room # in your hotel
Change TO 0000000 to the highest room # in your hotel, Tap Ok
Scroll down to No. 30 GUEST VALID
Double Tap the 00:00-00:00
on GUEST VALID change it to the (Military) times you want guests to be able to access this door.
(Ex: if you want them to be able to use it 6am to 10pm, enter 06:00-22:00)
(Ex: if you want 24/7 access as long as they have a key, enter 00:00-23:59)
(EX: if you don't want guess to access a room and only staff members, enter 00:00-00:01)
Tap Ok
Tap Download
(Make sure to wake up the lock sensors by inserting any card, leaving it there
and having it light up red first)
Press Ok
Tap Ok, Aim the Infrared USB at the lock

L7) Disabling Lost/Stolen Staff or Guest Keys

Create a new issue of the staff or guest key and run it by the appropriate door.

The old keys will no longer work on a lock that is activated by a new issue of keys.
