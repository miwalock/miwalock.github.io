Frequently Asked Questions for AL5H Locks:

L1) How to Reprogram a lock as a guest or common door?




Large Keypad w/ Slider (StandAlone Unit) Instrctions

Before doing any kind of programming, check the date and time on your device first
Press the On Button
Sign in w/ Staff Code and Password, Press F1(Ok)
Select 7 - DTU Setting, Press F1(Ok)
Select Date & Time Settings, Press Enter
Put in the correct Date and Time (Military Time)

You'll have to Detach the Keypad from Slider to be able to use it for programming locks

Programming a new guest door lock.
a) Turn off FDU.
b) Seprate the Keypad from the Base Slider.
c) xxx
d) Turn on the DTU.
e) Sign in with Staff Code and Password
f) Go to 1. for Normal Mode.
g) Go to 4. for Data Mode.
h) Room Type is 02 for guest and press F1.
i) Input the room number and press enter.
j) Input the floor number, aim unit at lock, insert any card into the lock to wake it up (Red light is ok) and press the SEND/RCV key.
k) At Hotel Code Error prompt press the Error Reset key.
**Note: if programming a brand new card reader, you MUST sign in with the Maintenance Code. It is normal to get the hotel code error, select Error Reset and then press SEND/RCV and Aim at lock again
l) Press F1 for Retry Download.
m) DTU will say “Now Loading” and “Loading End” when finished.
*Aim at the black stripe to the right of the indicator light (3-4 inches away)
*The Item Resel. Button is your back button if you need to go to a previous choice


Programming a new lock as a common door.
a) Follow steps a-g from above.
b) Room type is 01 common and press enter.
c) Press F1.
d) Press F4 to edit.
e) Input floor number and press down arrow.
f) Press enter to highlight days of the week, (all days should be highlighted).
g) Press F1.
h) Press down arrow till you get to Room Range 1.
i) Press F4 to edit.
j) Input lowest room in range and press enter.
k) Input highest room in range and press F1 for ok.
l) Press F1 for ok.
m) Aim unit at lock, insert any card into the lock to wake it up (Red light is ok), and press SEND/RCV.
n) At Hotel Code Error prompt press the Error Reset button.
**Note: if programming a brand new card reader, you MUST sign in with the Maintenance Code. It is normal to get the hotel code error, select Error Reset and then press SEND/RCV and Aim at lock again
o) Press F1 to Retry Download.
p) Aim DTU at lock and press the SEND/RCV key.
q) DTU will say “Now Loading” and “Loading End” when finished.

Small Keypad (legacy) Instructions
*Before Programming- Check the date/time by turning on, enter staff code/password, Press 1 (R/C), F2, F2, Enter the date on W-3 and (military) time on W-4 (hours,minutes,seconds). If on W-5 or higher, keep pressing enter to cycle through until you are back at W-3.


Programming a new guest door lock.
a) Press the on/off button to turn on DTU
b) Sign in with staff code and password then press Enter
c) Press the #1 button for R/C
d) Select F1 for Normal Mode.
e) Select F4 for Data Mode.
f) Room Type is 02 for guest doors, press Enter.
g) Input the room number and press enter three times.
h) Input the floor number and press enter three times.
i) Data Mode DLL , Push SEND/RCV button
j) DTU will say “Try Connect” insert card into the lock, don’t remove card aim DTU at LED until display shows “Loading Success” when finished, if Loading fail push error reset and try again
*Aim at the black stripe to the right of the indicator light (3-4 inches away)
*The Item Resel. Button is your back button if you need to go to a previous choice
**Note: if programming a brand new card reader, you MUST sign in with the Maintenance Code. It is normal to get the hotel code error, select Error Reset and then press SEND/RCV and Aim at lock again



Programming a new lock as a common door
a) Turn on the DTU.
b) Sign in with your staff code and password, then press Enter
c) Press the #1 button for R/C
d) Select F1 for Normal Mode.
e) Select F4 for Data Mode.
f) Room type is 01 common doors, press the up arrow one time to select, press Enter three times
g) Input a Room Number to identify this door, press Enter three times.
h) Input a Floor Number and press Enter three times.
i) Press Clear, then Input Room Range, type lowest room number and press Enter, then type highest room number and press Enter three times.
j) Press Enter to bypass Special Number
k) Press Enter to bypass Non-Guest Number
l) For Guest Valid Time- Using military time, enter an it as an 8 digit number (first 4 is start, last 4 is end). Ex: 06002359 means it is available to guests from 6AM to 11:59PM. Press Enter three times after.
m) Input days of the week, (by default 1234567 should be on screen). Press Enter.
n) If you want the door to stay open and not require a key during certain hours, you will need to set a Time Zone
n2) Otherwise skip Time Zone
o) To set a Time Zone, Press the down arrow to change it from Unset to Time Zone 1 and press enter
o2) Enter the start and end time- Example – 06002200 and press Enter three times, that will set the lock to stay unlocked from 6AM to 10PM and allow free access during that time
p) Insert any card to wake up the sensors and have it light up red, Aim unit at lock and press SEND/RCV, Lock can only be programmed while it is awake/red light
q) DTU will say “Try Connect” and “Loading Success” when finished.
*Aim at the black stripe to the right of the indicator light (3-4 inches away)
*The Item Resel. Button is your back button if you need to go to a previous choice
**Note: if programming a brand new card reader, you MUST sign in with the Maintenance Code. It is normal to get the hotel code error, select Error Reset and then press SEND/RCV and Aim at lock again

Note: If you want to disable guests and only allow staff to enter a room:
Program it as a common door, but on step L, enter 00000001

Touch Screen Tablet Instructions
Plug in the Black Actisys USB, that is where the signal comes out of
Check the Windows date and time on your tablet before opening the DTU software
Double Click DTU Starter Icon
Tap Ok [F1] Button
Enter Staff Code and Password
Tap Ok
Check the Date and Time on the DTU Software before doing any kind of programming

For Guests Room:
a) Select 1. Lock Setup Menu
b) Select 1. Initialize Lock Data
c) For Room Type Enter 2
d) Enter the Room #
e) Enter the Floor # (Example: for first floor, enter 01) then press Ok
f) Press Download
g) (Make sure to wake up the lock sensors by inserting any card, leaving it there and having it LIGHT UP RED while you send the signal)
h) Press Ok
i) Aim the Infrared USB at the lock

For Common Doors:
After Entering Staff Code and Password-
a) Select 1. Lock Setup Menu
b) Select 1. Initialize Lock Data
c) For Room Type Enter 1
For Room # (Depends on which floor it is. If first floor put in 101, or else second floor- 201, etc)
(Room # isn't important for common doors, you can have multiple common doors w/ same #)
d) Then Press Ok Enter the Floor # (Example for first floor enter 01)
e) Double Tap the numbers to the right of room range 1
f) On Room Range 1, change FROM 000000 to the lowest room # in your hotel
g) Change TO 0000000 to the highest room # in your hotel, Tap Ok
Scroll down to No. 30 GUEST VALID
Double Tap the 00:00-00:00
on GUEST VALID change it to the (Military) times you want guests to be able to access this door.
(Ex: if you want them to be able to use it 6am to 10pm, enter 06:00-22:00)
(Ex: if you want 24/7 access as long as they have a key, enter 00:00-23:59)
(EX: if you don't want guess to access a room and only staff members, enter 00:00-00:01)
h) Tap Ok
i) Tap Download
j) (Make sure to wake up the lock sensors by inserting any card, leaving it there and having it LIGHT UP RED While you send the signal)
k) Press Ok
l) Tap Ok, Aim the Infrared USB at the lock


L2) Lock lights up red and doesn't allow access to the room.

(under construction)
Reprogram the lock using your DTU lock programmer (there's 3 kinds, choose the one you have), see Question L1 above

If it lights up red, then yellow, it is a sequence issue, try clearing the sequence, then using a card again
See question L (clearing the sequence) below

For guest key still lighting up red, try making a guest key with the check in date of week ago, and check out of 1 week from from today, then using it on the lock
If you get it to work it may be a date/time issue.
For staff keys not working, try making a New Issue under submaster card and using it on the lock

If it is still not working with any card, you may have to replace the card reader.


L3) Lock is out of power and unresponsive to any card

OR

Plug Emergency Battery kit undernearth lock and use key card to gain entry
or
Use the Physical Override Key if your lock has that feature

Replace the lock's batteries once room is accessed, and Reprogram the lock to correct date/time.
Try any card after and see if it lights up green.

If it is still unresponsive after replacing the batteries, you'll have to replace the card reader inside.
You can call our office if you need to order another one.

Instructions for LARGE KEYPAD & SLIDER STANDALONE Keymaker Only (old school type) -
Adding staff members to the Large Keypad StandAlone Unit
a) Make sure the Large Keypad is attached to the Slider
b) xxx
c) Turn on FDU.
d) Enter Staff Code and press enter
e) Enter Password and press F1
f) Press the #2 button for Default setup.
g) Press the #2 button for Master File Menu.
h) Press the #6 button for Sub-Master Setup.
i) Press F1 to Add a New Sub-Master Level.
j) At SMS Group Name prompt Press the F3 button to bring up the character keyboard and use the up and down arrows to scroll to the selected letter and press enter.
k) After entering SMS Group name press the F3 button again to disengage keyboard.
l) Press the Down arrow 2 times to highlight SMS Level Number and input a number between 1-99 then press enter, the group number will always stay 01.
m) Press Enter at group number.
n) Input Starting month and press enter, input starting day and press enter, input starting year and press enter, input starting time and press enter.
o) Input the ending month and press enter, input the ending day and press enter, input ending year and press enter, input ending time and press enter.
p) Highlight all valid days of the week by using arrows to navigate to the days and the enter button to highlight them.
q) Press the F2 button for option 1.
r) Press the enter button to chose the command. Use up/dwn arrows to navigate list, highlight the selected command and press enter again, then press the down arrow to go to parameter 1.
s) Input parameter 1 then down arrow to parameter 2 and input.
t) Press the F2 button to add.
u) Press the F1 button for ok.
v) Press the F1 button for ok.

Issuing Staff keys with the Large Keypad StandAlone Unit
a) Select Management Card Menu.
b) Press 3 for Issue Sub-Master Key.
c) Use up/dwn arrows to scroll to level number of the key that you want to make and press the F1 button for ok.
d) Use the check screen to confirm the data.
e) Press F1 to issue.
f) Use up/dwn arrow to scroll the staff member authorizing the production of this key and press f4 to select.
g) Press F1 to issue.
h) Insert a Smart Chip Card on top.
